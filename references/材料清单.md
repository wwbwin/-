# 理赔材料清单

按案件类型和险种分类的材料清单，参考《商业车险示范产品理赔实务要点（2020试行版）》第八节。

---

## 一、通用材料（所有案件都需要）

- [ ] 被保险人身份证
- [ ] 银行账户信息（领款用，符合反洗钱规定）
- [ ] 索赔申请书
- [ ] 交通事故认定书（或事故证明）

---

## 二、车损险案件材料

### 基本材料

- [ ] 被保险机动车行驶证
- [ ] 驾驶员驾驶证
- [ ] 驾驶员身份证
- [ ] 事故现场照片

### 维修案件

- [ ] 维修发票（增值税发票）
- [ ] 维修清单
- [ ] 车辆损失情况确认书（定损单）
- [ ] 配件更换清单（如有）

### 推定全损案件

- [ ] 推定全损协议
- [ ] 车辆报废证明（如有）
- [ ] 车辆残值评估报告

### 施救费用

- [ ] 施救费用发票
- [ ] 施救清单
- [ ] 施救单位资质证明

---

## 三、第三者责任保险案件材料

### 基本材料

- [ ] 第三者的身份证明
- [ ] 第三者的损失清单
- [ ] 第三方车辆行驶证、驾驶证（如涉及车辆）

### 财产损失

- [ ] 财产损失清单
- [ ] 财产损失照片
- [ ] 维修发票或赔偿凭证

### 人身伤亡

- [ ] 受害者身份证
- [ ] 医疗费用发票
- [ ] 诊断证明
- [ ] 住院小结、出院小结
- [ ] 伤残鉴定报告（如涉及伤残）
- [ ] 误工证明、收入证明
- [ ] 护理证明、护理费发票
- [ ] 交通费发票
- [ ] 死亡证明、殡葬证明（涉及死亡）

---

## 四、车上人员责任保险案件材料

### 驾驶员

- [ ] 驾驶员驾驶证
- [ ] 诊断证明、医疗发票
- [ ] 住院小结

### 乘客

- [ ] 乘客身份证明
- [ ] 乘客户籍证明
- [ ] 医疗费用发票
- [ ] 诊断证明、住院小结
- [ ] 伤残鉴定报告（如涉及伤残）

---

## 五、代位求偿案件额外材料

- [ ] "代位求偿"案件索赔申请书
- [ ] 责任对方的车牌号
- [ ] 责任对方交强险承保公司
- [ ] 责任对方商业三者险承保信息
- [ ] 已从第三方获得赔偿的凭证

---

## 六、特殊案件材料

### 案件类型 | 额外材料 |

|---------|----------|
| 酒驾/醉驾案件 | 血液检测报告、酒驾鉴定报告 |
| 逃逸案件 | 交警事故认定书 |
| 涉水案件 | 气象证明、积水证明 |
| 盗抢案件 | 立案证明、车辆登记证书 |

---

## 七、材料提交方式

根据《实务要点》第八节，提交方式包括：

1. **现场提交**：理赔人员直接收取
2. **柜面提交**：各服务网点
3. **快递提交**：小额简易案件
4. **线上提交**：微信、APP提交电子材料

---

## 八、注意事项

1. 所有材料应当真实、完整
2. 复印件需与原件核对
3. 保险公司只能简化材料，不能增加材料要求
4. 责任明确的材料不齐时，应一次性告知客户补充
5. 鼓励使用电子化单证
