# 商业车险赔款计算公式汇总

基于《商业车险示范产品理赔实务要点（2020试行版）》第九节理算部分整理。

---

## 一、理算顺序

**先交强险，后商业险**

---

## 二、机动车损失保险（车损险）

### 1. 全部损失

```
车损险总赔款 = (车损赔款 + 施救费赔款 - 绝对免赔额) × (1 - 绝对免赔率)

车损赔款 = (保险金额 - 交强险应赔付本车损失金额) × 被保险车辆事故责任比例
施救费赔款 = (核定施救费 - 交强险应赔付本车施救费金额) × 被保险车辆事故责任比例
```

### 2. 部分损失

```
车损险总赔款 = (车损赔款 + 施救费赔款 - 绝对免赔额) × (1 - 绝对免赔率)

车损赔款 = (实际修复费用 - 交强险应赔付本车损失金额) × 被保险车辆事故责任比例
施救费赔款 = (核定施救费 - 交强险应赔付本车施救费金额) × 被保险车辆事故责任比例
```

### 3. 代位求偿方式

```
车损险总赔款 = (车损赔款 + 施救费赔款 - 绝对免赔额) × (1 - 绝对免赔率)

车损赔款 = 保险金额 - 被保险人已从第三方获得的车损赔偿金额
施救费赔款 = 核定施救费 - 被保险人已从第三方获得的施救费赔偿金额
```

## 三、第三者责任保险（三者险）

### 计算公式

```
当 (核定第三者损失金额 - 交强险分项赔偿限额) × 事故责任比例 ≥ 每次事故赔偿限额时：
  赔款 = 每次事故赔偿限额 × (1 - 绝对免赔率)

当 (核定第三者损失金额 - 交强险分项赔偿限额) × 事故责任比例 < 每次事故赔偿限额时：
  赔款 = (核定第三者损失金额 - 交强险分项赔偿限额) × 事故责任比例 × (1 - 绝对免赔率)
```

### 主挂车理赔（2020版示范条款）

```
主车应赔款 = 总赔款 × [主车责任限额 ÷ (主车责任限额 + 挂车责任限额)]
挂车应赔款 = 总赔款 × [挂车责任限额 ÷ (主车责任限额 + 挂车责任限额)]
```

总赔款限额 = 主车与挂车三者险责任限额之和

## 四、车上人员责任保险

### 计算公式（每座分别计算）

```
当 (核定每座人身伤亡损失金额 - 交强险应赔偿金额) × 事故责任比例 ≥ 每次事故每座赔偿限额时：
  每座赔款 = 每次事故每座赔偿限额 × (1 - 绝对免赔率)

当 (核定每座人身伤亡损失金额 - 交强险应赔偿金额) × 事故责任比例 < 每次事故每座赔偿限额时：
  每座赔款 = (核定每座人身伤亡损失金额 - 交强险应赔偿金额) × 事故责任比例 × (1 - 绝对免赔率)

每次事故总赔款 = 每座受害人赔款之和
```

## 五、附加险计算

### 1. 车轮单独损失险

```
当 (核定车轮损失金额 - 交强险赔偿金额) × 事故责任比例 ≥ 保险金额时：
  赔款 = 保险金额

当 (核定车轮损失金额 - 交强险赔偿金额) × 事故责任比例 < 保险金额时：
  赔款 = (核定车轮损失金额 - 交强险赔偿金额) × 事故责任比例
```

⚠️ 注意：累计赔款达到保险金额时，本附加险终止

### 2. 车身划痕损失险

```
当实际修复费用 ≥ 保险金额时：
  赔款 = 保险金额

当实际修复费用 < 保险金额时：
  赔款 = 实际修复费用
```

⚠️ 注意：累计赔款达到保险金额时，本附加险终止

### 3. 修理期间费用补偿险

```
全部损失：赔款 = 日补偿金额 × 约定最高补偿天数

部分损失：
  - 补偿天数 = min(约定修理天数, 实际修理天数)
  - 赔款 = 日补偿金额 × 补偿天数（不超过最高补偿天数）
```

### 4. 车上货物责任险

```
当 (核定货物损失金额 - 交强险对车上货物赔款) × 事故责任比例 ≥ 责任限额时：
  赔款 = 责任限额

当 (核定货物损失金额 - 交强险对车上货物赔款) × 事故责任比例 < 责任限额时：
  赔款 = (核定货物损失金额 - 交强险对车上货物赔款) × 事故责任比例
```

### 5. 精神损害抚慰金责任险

```
赔款 = 法院判决的精神损害赔偿责任 - 交强险精神损害赔款
（最高不超过责任限额）
```

### 6. 法定节假日限额翻倍险

适用于三者险，保额翻倍：

```
当 核定第三者损失金额 × 事故责任比例 ≥ 2倍主险限额时：
  赔款 = 主险每次事故赔偿限额

当 主险限额 ≤ 核定第三者损失金额 × 事故责任比例 < 2倍主险限额时：
  赔款 = (核定第三者损失金额 × 事故责任比例 - 主险每次事故赔偿限额)

当 核定第三者损失金额 × 事故责任比例 < 主险限额时：
  赔款 = 0
```

### 7. 医保外医疗费用责任险

对超出医保范围的医疗费用进行赔付，最高不超过附加险赔偿限额。

---

## 六、关键术语解释

| 术语 | 含义 |
|------|------|
| 保险金额 | 保单载明的最高赔偿限额 |
| 实际修复费用 | 保险人与被保险人协商确定的修复费用 |
| 绝对免赔率 | 投保时约定的免赔百分比 |
| 绝对免赔额 | 每次事故绝对免赔的金额 |
| 事故责任比例 | 交警认定或保险公司核定的责任比例 |
| 代位求偿 | 保险公司先赔付后向责任方追偿 |
