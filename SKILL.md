# 🚗 商业车险理赔助手

> 基于官方文档《商业车险示范产品理赔实务要点（2020试行版）》

---

## 📖 简介

中国大陆地区车主发生道路交通事故后，可通过本助手获得专业的车险理赔指导。

**本助手不提供保险销售服务**，仅基于行业协会公开的理赔实务要点，为车主解答理赔流程中的常见问题。

---

## 🎯 核心功能

1. **理赔流程咨询** - 事故发生后各环节操作要点
2. **理赔材料清单** - 按案件类型（车损/人伤/三者）提供材料清单
3. **赔款计算指导** - 车损险、三者险、车上人员险的赔款计算公式
4. **拒赔风险提示** - 常见拒赔情形及注意事项
5. **时效与权益** - 法定的理赔时效与车主权利

---

---

## 🔄 理赔流程

```
出险报案 → 调度派工 → 查勘现场 → 立案估损 → 定损 → 核价核损 → 人伤处理 → 资料收集 → 理算 → 核赔 → 赔款支付 → 结案归档
```

## ❓ 常见问答

### 出险报案

**Q: 出险后应该第一时间做什么？**
- 立即向保险公司报案（365天×24小时渠道畅通）
- 保护事故现场，及时向交警报案
- 涉及人员伤亡的拨打120
- 记录报案号、理赔人员联系方式

**Q: 报案时需要提供哪些信息？**
- 保单信息（车牌号、车架号）
- 事故经过、时间、地点
- 驾驶员信息、驾驶证

### 查勘定损

**Q: 查勘人员多久能到现场？**
- 城市城区：45分钟内
- 城市郊区：1小时内
- 城市市辖县：2小时内

**Q: 定损完成后还能追加定损吗？**
- 原则上应一次性定损完成
- 维修中若发现新损失，经核实后可追加
- 必须是本次保险事故造成的损失

### 赔款计算

**Q: 车损险赔款怎么计算？**

```
车损险总赔款 = (车损赔款 + 施救费赔款 - 绝对免赔额) × (1 - 绝对免赔率)
车损赔款 = (实际修复费用 - 交强险应赔付本车损失金额) × 事故责任比例
```

**Q: 三者险赔款怎么计算？**

```
赔款 = (核定第三者损失 - 交强险分项赔偿限额) × 事故责任比例 × (1 - 绝对免赔率)
```

**Q: 交强险和商业险的理赔顺序？**
- 先交强险，后商业险
- 交强险分项赔偿：有责/无责分限额

### 拒赔应对

**Q: 哪些情况容易被拒赔？**
- 酒后驾驶、吸毒驾驶
- 无证驾驶、准驾不符
- 事故后擅自移动现场
- 未及时报案
- 改装车辆导致危险程度增加未通知保险公司

**Q: 收到拒赔通知书怎么办？**
- 保险公司必须在核定后3日内发出《拒赔通知书》
- 可以要求保险公司说明拒赔依据
- 可以申请复议或提起诉讼

### 材料清单

| 案件类型 | 核心材料 |
|---------|---------|
| 车损案件 | 行驶证、驾驶证、��修发票、事故认定书 |
| 人伤案件 | 诊断证明、医疗发票、伤残鉴定报告 |
| 三者案件 | 第三方损失清单、赔偿协议 |

详见 `references/材料清单.md`

### 时效提醒

- 报案：出险后及时报案（48小时内最佳）
- 定损：核损通过后及时告知客户最终金额
- 赔款：收到材料后，按《保险法》时效支付
- 拒赔：核定后3日内发出拒赔通知书
