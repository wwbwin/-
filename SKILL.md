## 📖 前言

为了让中国大陆地区车险销售人员更清晰的理解汽车商业险的理赔流程，特别制作了此内容。我在国内ZA保险公司任职，是行业从业人员，后面将在后续版本中逐步更新，此次更新为2026年4月17日，北京。

后续会增加更多的功能，让这个小助手随时随地的帮助到更多人。

---

基于中国保险行业协会《商业车险示范产品理赔实务要点（2020试行版）》，为用户提供专业的商业车险理赔指导。

## 核心能力

1. **理赔流程咨询** - 解答各环节操作要点
2. **理赔材料清单** - 按险种和案件类型提供完整材料清单
3. **赔款计算指导** - 车损险、三者险、车上人员险、附加险的赔款计算公式
4. **拒赔风险提示** - 常见拒赔情形及规避建议
5. **权益保护指导** - 帮用户理解自身权利，对抗不合理拒赔

## 理赔流程总览

```
出险报案 → 调度派工 → 查勘现场 → 立案估损 → 定损 → 核价核损 → 人伤处理 → 资料收集 → 理算 → 核赔 → 赔款支付 → 结案归档
```

## 常用场景问答

### 🔔 出险报案

**Q: 出险后应该第一时间做什么？**
- 立即报案，保险公司要求"365天×24小时"报案渠道畅通
- 记录报案电话、报案号、理赔人员联系方式
- 保护事故现场，及时向交警报案
- 涉及人员伤亡的拨打120

**Q: 报案时需要提供哪些信息？**
- 保单信息、报案人身份
- 事故经过、时间、地点
- 车辆信息（车牌号、车架号）
- 驾驶员信息、驾驶证

### 🔧 查勘定损

**Q: 查勘人员多久能到现场？**
- 城市城区：45分钟内
- 城市郊区：1小时内
- 城市市辖县：2小时内

**Q: 定损完成后还能追加定损吗？**
- 原则上应一次性定损完成
- 维修过程中发现的新损坏部件，经核实后可追加定损
- 注意：必须是本次保险事故造成的损失

### 💰 赔款计算

**Q: 车损险赔款怎么计算？**

参考 `references/赔款计算公式.md`：

```
车损险总赔款 = (车损赔款 + 施救费赔款 - 绝对免赔额) × (1 - 绝对免赔率)
车损赔款 = (实际修复费用 - 交强险应赔付本车损失金额) × 被保险车辆事故责任比例
```

**Q: 三者险赔款怎么计算？**

```
赔款 = (核定第三者损失金额 - 交强险分项赔偿限额) × 事故责任比例 × (1 - 绝对免赔率)
```

**Q: 交强险和商业险的理赔顺序？**
- 先交强险，后商业险
- 交强险分项赔偿：有责/无责分限额

### 🚫 拒赔应对

**Q: 哪些情况容易被拒赔？**
- 酒后驾驶、吸毒驾驶
- 无证驾驶、准驾不符
- 事故后擅自移动现场
- 未及时报案
- 改装车辆导致危险程度增加未通知保险公司
- 车辆使用性质改变

**Q: 收到拒赔通知书怎么办？**
- 保险公司必须在核定后3日内发出《拒赔通知书》
- 可以要求保险公司说明拒赔依据
- 可以申请复议或提起诉讼
- 做好证据保全，准备应诉

### 📋 材料清单

根据案件类型，所需材料不同：

| 案件类型 | 核心材料 |
|---------|---------|
| 车损案件 | 行驶证、驾驶证、维修发票、交通事故认定书 |
| 人伤案件 | 诊断证明、医疗发票、伤残鉴定报告 |
| 三者案件 | 第三方损失清单、赔偿协议 |

详见 `references/材料清单.md`

### ⏱️ 时效提醒

- 报案：出险后及时报案（48小时内���佳）
- 定损：核损通过后及时告知客户最终金额
- 赔款：收到材料后，按《保险法》时效支付
- 拒赔：核定后3日内发出拒赔通知书

## 争议处理

遇到理赔争议时：
1. **协商和解** - 与保险公司协商
2. **投诉** - 向保险公司总部或银保监会投诉
3. **调解** - 申请行业协会调解
4. **仲裁/诉讼** - 通过仲裁或法院诉讼解决
